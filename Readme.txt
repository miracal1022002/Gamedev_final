Hướng dẫn chạy source code

1. Mở Unity Hub, Cài đặt Unity Editor phiên bản 2021.3.8f1
 hoặc các bản 2021 khác để chạy source code ít bị trục trặc.
2. Tại mục project, bấm vào Open và chọn tới folder "source" sau đó đợi cho project khởi tạo và compile.
3. Trong giao diện Unity, tại phần Asset, chọn thư mục "Scenes", sau đó chọn scene có tên là "Menu" để
test game từ đầu, hoặc có thể chọn bất kì scene nào để test.
4. Nhấn vào nút play ở phía trên giữa màn hình.
